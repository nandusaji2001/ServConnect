# Chatbot Color Update - Green Theme

## Changes Made

The chatbot color scheme has been updated from purple to a fresh, modern green theme.

### Color Palette

**Primary Green Gradient:**
- Start: `#10b981` (Emerald 500)
- End: `#059669` (Emerald 600)

**Shadow Colors:**
- Light: `rgba(16, 185, 129, 0.4)`
- Dark: `rgba(16, 185, 129, 0.6)`

### Updated Components

1. **Toggle Button** - Green circular button with gradient
2. **Header** - Green gradient header bar
3. **Message Avatar** - Green circular avatar for bot messages
4. **User Messages** - Green gradient background
5. **Navigation Buttons** - Green gradient with hover effects
6. **Send Button** - Green circular button
7. **Input Focus** - Green border on focus
8. **Typing Animation** - Green dots during typing
9. **Suggestion Hover** - Green border and text on hover

### Customization

To change to a different color, edit `chatbot.css` and replace:

```css
/* Current green theme */
background: linear-gradient(135deg, #10b981 0%, #059669 100%);

/* Example alternatives: */

/* Blue theme */
background: linear-gradient(135deg, #3b82f6 0%, #2563eb 100%);

/* Purple theme (original) */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* Orange theme */
background: linear-gradient(135deg, #f97316 0%, #ea580c 100%);

/* Pink theme */
background: linear-gradient(135deg, #ec4899 0%, #db2777 100%);

/* Teal theme */
background: linear-gradient(135deg, #14b8a6 0%, #0d9488 100%);
```
